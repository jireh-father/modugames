## 3. 플레이어가 광고를 "자발적으로" 보는 순간 TOP 8

| 순위 | 트리거 | 작동 방식 | 왜 효과적인지 |
|---|---|---|---|
| 1 | **부활/컨티뉴** | 죽었을 때 광고 보면 이어서 플레이 | 손실 회피 심리 - 투자한 시간 아까움 |
| 2 | **코인 2배/3배** | 클리어 후 광고 보면 보상 배로 | "공짜 돈" 느낌, 방금 성공해서 기분 좋은 상태 |
| 3 | **추가 목숨/이동** | 목숨/이동 횟수 소진 시 광고로 추가 | 기다리지 않고 바로 플레이 가능 |
| 4 | **캐릭터/스킨 해금** | 광고로 화폐 벌어서 캐릭터 수집 | 수집 욕구 + 자기표현 |
| 5 | **대기 시간 스킵** | 광고 보면 쿨다운 즉시 해제 | 30초 광고로 몇 시간 절약 |
| 6 | **힌트/단서** | 퍼즐에서 막힐 때 광고로 힌트 | 좌절감 해소 + 실질적 가치 |
| 7 | **레벨 스킵** | 어려운 레벨 광고로 건너뛰기 | 이탈 방지 (최후의 수단) |
| 8 | **파워업/부스트** | 게임 시작 전 광고로 버프 받기 | 전략적 선택 느낌 |

> 핵심: **상황에 맞는 보상**을 제공해야 함. 죽었으면 → 부활, 이겼으면 → 코인 2배

## 4. 최적 광고 빈도 & 배치 전략

### Interstitial (전면 광고)
- 세션당 **1~3회**
- **3~5분 간격**
- 절대 게임 중간에 노출 금지 → 레벨 클리어, 메뉴 복귀 등 자연스러운 전환점에만
- 신규 유저에게는 더 적게 (세션당 1회)

### Rewarded Video (보상형 광고)
- 세션당 **3~5회 기회** (자발적 선택이므로 빈도 높아도 OK)
- 같은 보상 타입은 하루 2~3회 제한 (예: 부활 광고 하루 3번까지)

### Banner
- 상시 노출이지만 eCPM이 너무 낮음 ($0.45~$0.68)
- 최근 탑 게임들은 배너 제거 추세

### 장르별 추천 믹스

| 장르 | 추천 광고 조합 |
|---|---|
| 퍼즐 | 보상형 광고 Heavy + 3~4레벨마다 전면 광고 |
| 액션/아케이드 | 보상형 광고 위주 + 세션 종료 시 전면 광고 |
| 방치형/시뮬레이션 | 거의 보상형 광고만 (시간 스킵, 생산 부스트) |
| 하이퍼캐주얼 | 전면 광고 + 보상형 균형 (세션이 매우 짧으므로) |

## 6. 중독성 있는 캐주얼 게임의 7가지 원칙

### 1. 심플함 + 깊이
- 원탭 메카닉, 한 화면, 즉시 이해
- 배우기 쉽고 마스터하기 어렵게
- 예: Flappy Bird (탭으로 날기), Crossy Road (탭으로 건너기)

### 2. 즉각적 보상
- 모든 액션에 시각/청각 피드백
- 초반 레벨은 의도적으로 쉽게 → 성취감 유도
- 화려한 이펙트, 파티클, 화면 흔들림 ("Juice")

### 3. 강제 루프 (심리적 Hook)
- 액션 → 보상 → 동기 → 반복
- 랜덤 보상(가챠)이 예측 가능한 보상보다 중독성 높음

### 4. 손실 회피
- 스트릭 보상 놓치면 리셋
- 부활 광고의 핵심 원리: 이미 투자한 진행도를 잃기 싫은 심리

### 5. 점진적 난이도 ("Flow State")
- 너무 쉬움(지루함)과 너무 어려움(좌절감) 사이의 "존"
- 2025 트렌드: AI 기반 동적 난이도 조절 → D3 리텐션 18% 향상

### 6. 단기 + 장기 목표
- **코어 루프** (단기): 이 레벨 클리어, 이 점수 돌파
- **메타 루프** (장기): 모든 캐릭터 수집, 최고 리그 도달

### 7. "Just One More" 디자인
- 3~5분 "스낵" 세션, 하루 3~5회 반복
- 세션 사이 이유 제공: 에너지 충전, 데일리 보상, 친구 도전

## 7. 성공 사례 심층 분석

### Case 1: Crossy Road - 보상형 광고의 원조

```
수익 모델:
├── Rewarded Video → 코인 획득 → 캐릭터 뽑기기 (랜덤)
├── $0.99 직접 캐릭터 구매
├── $3.99 캐릭터 + 코인 2배 번들
└── 에너지 시스템 없음, 제한 없이 무한 플레이

핵심: "공정한 수익화 시스템 - 완전히 무시해도 되지만,
       그래도 사람들이 자발적으로 광고를 봄"

매출: 90일 만에 $1000만 (광고 $300만 포함)
```

### Case 2: Subway Surfers - 12년간 살아남은 비결

```
수익 모델:
├── Rewarded Video → 부활, 코인 배수, 미스터리 박스
├── Interstitial → 홈 버튼 복귀 등 자연 전환점에만
├── IAP → 캐릭터, 코인 팩 (선택적, Pay-to-win 아님)
└── 광고 최적화 전담팀: 미디에이션 워터폴 + A/B 테스트

리텐션 비결:
├── "World Tour" 시스템 → 몇 주마다 도시/테마 업데이트
├── 시퀄 안 만듦 → 유저 분산 방지 (Temple Run의 실수 반면교사)
└── 12년간 꾸준한 콘텐츠 업데이트

성과: 월 10억+ 광고 노출, ARPDAU 67% YoY 증가
```

### Case 3: Block Blast - 2025 최강자

```
수익 모델:
├── 순수 광고 (배너 + 전면)
├── IAP 없음에도 하루 $100만 수익
└── 70M DAU, 300M MAU

성공 비결:
├── 연간 10,000+ A/B 테스트 (하루 300+)
├── 자체 "Block AI" 행동 시뮬레이션 플랫폼
├── TikTok 중심 UA (90% TikTok 유입)
└── 극도의 데이터 드리븐 레벨 디자인
```

### Case 4: Flappy Bird - 솔로 개발자의 전설

```
수익: 배너 광고만으로 피크 하루 $50,000
교훈: 만약 Rewarded/Interstitial을 썼다면 10~50배 더 벌었을 것
      → 심플한 수익화의 한계 vs 잠재력 둘 다 보여줌
```

### Case 5: Among Us - 2년 후 바이럴 폭발

```
출시: 2018년 → 무반응 → 2020년 갑자기 폭발
바이럴 요인:
├── 스트리머 Sodapoppin이 4시간 방송 → 연쇄 반응
├── 본질적으로 소셜 (4~10인 필수)
├── "스트리밍하기 좋은" 게임 (속임수, 드라마, 배신)
├── "sus" 밈이 일상 용어로 진입
├── COVID 타이밍: 비대면 소셜 니즈
└── 마케팅 비용: $0 (전부 오가닉)

매출: ~$1.12억, DAU 6000만 달성
```

## 10. 우리 게임에 적용할 액션 아이템

### 추천 방향 요약

1. **장르**: Sort Puzzle, Block Puzzle (3세대), 또는 Cross-Genre Fusion → 포화되지 않은 영역
2. **수익 모델**: 하이브리드 (Rewarded Video 중심 + 가끔 Interstitial + 선택적 IAP)
3. **메타 시스템**: 데일리 보상, 캐릭터 수집, 시즌패스, 리더보드
4. **세션 디자인**: 3~5분 완결 루프, 하루 3~5회 반복 가능
5. **코어 루프**: 원탭/원스와이프 메카닉, 즉각적 피드백, 점진적 난이도
6. **UA 전략**: TikTok 중심 (Block Blast의 90%가 TikTok에서 유입)
7. **A/B 테스트**: 광고 빈도, 배치, 보상 타입, 난이도 커브 모두 데이터 기반 결정
8. **타겟**: Android 먼저 (광고 수익 63%), 이후 iOS 확장

### ARPU 목표치
- 순수 광고: ARPDAU $0.05~$0.15
- 하이브리드 (광고+IAP): ARPDAU $0.10~$0.30
- DAU 100만 달성 시: 하루 $50,000~$300,000 수익 가능
