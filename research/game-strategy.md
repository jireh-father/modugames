## 의사결정 프레임워크

게임을 선택할 때 아래 5가지를 모두 만족해야 함:

| 기준 | 왜 중요한지 |
|---|---|
| 1. 내가 코드로 구현 가능한가 | Godot + GDScript로 1~2인 개발 가능해야 함 |
| 2. 시장이 성장 중인가 | 포화 시장 진입은 자살행위 |
| 3. 차별화 포인트가 명확한가 | "왜 이 게임을 깔아야 하는지" 한 문장 |
| 4. 광고 수익 모델이 자연스러운가 | 보상형 광고가 게임 흐름에 녹아들어야 함 |
| 5. TikTok에서 5초 클립이 매력적인가 | 오가닉 UA의 핵심 |

## 후보 3개 평가

### 후보 A: 🧊 블록 퍼즐 + 정원 꾸미기

```
코어 루프: Block Blast 스타일 블록 배치 → 줄 클리어
메타 루프: 클리어할 때마다 정원에 꽃/나무가 자람
차별화:   "내가 퍼즐을 풀수록 정원이 예뻐진다"
```

| 항목 | 점수 | 이유 |
|---|---|---|
| 구현 가능성 | ★★★★☆ | 그리드 로직은 검증됨, 정원 시스템 추가 필요 |
| 시장 성장성 | ★★★★★ | 블록퍼즐 = 2025 #1 (3.56억 다운로드) |
| 차별화 | ★★★☆☆ | Block Blast 대비 차별화 약함, 후발주자 |
| 광고 자연스러움 | ★★★★☆ | 부활, 블록 미리보기, 정원 아이템 |
| 바이럴 잠재력 | ★★★☆☆ | 정원이 예쁘면 공유 가능하나, 클립이 약함 |
| **종합** | **19/25** | 안전하지만 뻔함. Block Blast와 정면충돌 위험 |

### 후보 B: 🐱 고양이 카페 소트 퍼즐

```
코어 루프: 색깔별 고양이를 맞는 쿠션/집으로 분류
메타 루프: 고양이 카페 꾸미기, 희귀 고양이 수집
차별화:   "고양이 + 정렬 퍼즐 + 카페 경영"
```

| 항목 | 점수 | 이유 |
|---|---|---|
| 구현 가능성 | ★★★★☆ | 소트 로직 단순, 고양이 애니메이션 작업량 |
| 시장 성장성 | ★★★★★ | 소트퍼즐 YoY 5.6배 성장 |
| 차별화 | ★★★★☆ | 고양이 테마는 검증됨, 카페 메타는 매력적 |
| 광고 자연스러움 | ★★★★☆ | 힌트, 추가 이동, 희귀 고양이 뽑기 |
| 바이럴 잠재력 | ★★★★☆ | 귀여운 고양이 = 공유 의지 높음 |
| **종합** | **21/25** | 강력하지만, 고양이 게임 자체는 많음 |

### 후보 C: 🍡 말랑소트 (Mochi Sort) - 말랑말랑 모찌 정렬 퍼즐

```
코어 루프: 말랑말랑한 모찌(떡)를 색깔별로 접시에 정렬
메타 루프: 모찌 레시피 수집, 떡집 꾸미기, 캐릭터 수집
차별화:   "세상에서 가장 말랑말랑한 퍼즐게임"
```

| 항목 | 점수 | 이유 |
|---|---|---|
| 구현 가능성 | ★★★★★ | 소트 로직 단순 + 말랑 물리는 Tween/Shader로 구현 |
| 시장 성장성 | ★★★★★ | 소트퍼즐 YoY 5.6배 성장 |
| 차별화 | ★★★★★ | 모찌 물리감 = 시장에 없음, 촉감적 만족감 |
| 광고 자연스러움 | ★★★★★ | 힌트, 추가 접시, 특별 모찌, 레시피 해금 |
| 바이럴 잠재력 | ★★★★★ | ASMR + 말랑말랑 + 귀여운 얼굴 = TikTok 최적 |
| **종합** | **25/25** | 최고 점수. 성장시장 + 독보적 차별화 + 바이럴 |

## 최종 선택: 🍡 말랑소트 (Mochi Sort)

### 왜 이 게임인가

```
성장 시장     소트 퍼즐이 2025년 가장 빠르게 성장 (YoY 5.6배)
              아직 초기 → 진입 기회 충분
              38개 신작이 100만 다운로드 돌파

차별화        기존 소트 퍼즐은 전부 "공/물/나사" = 딱딱하고 무미건조
              모찌는 말랑말랑, 귀여운 얼굴, 음식 테마 = 완전히 다른 느낌
              "만지고 싶은 퍼즐" = 현존하는 소트 퍼즐에 없는 경험

바이럴        말랑말랑 물리 + ASMR 사운드 = "oddly satisfying"
              TikTok에서 ASMR 콘텐츠 시청시간 1.7배 높음
              5초 클립만으로도 "저게 뭐지? 깔아봐야지" 유도 가능

타겟 일치     36세 여성, 릴랙싱 목적, 5분 세션 = 말랑한 모찌와 완벽 매치
              음식 테마 + 귀여운 디자인 = 여성 유저 선호도 최상위

구현 가능     코어 메카닉: 컨테이너 간 아이템 이동 (검증된 로직)
              말랑 물리: Tween 애니메이션 + 간단한 Shader
              복잡한 물리엔진 불필요
```

## 게임 컨셉 상세

### 한 줄 소개
> **"세상에서 가장 말랑말랑한 퍼즐 게임"**

### 세계관
- 모찌 마을에서 떡집을 운영하는 캐릭터 "모찌"
- 각 레벨 = 고객의 주문을 맞추기 위해 모찌를 색깔별로 정리
- 주문 완성 → 고객 만족 → 코인 획득 → 떡집 업그레이드

### 마스코트: "모찌" (Mochi)

```
디자인 원칙:
├── 120x120px 앱 아이콘에서도 인식 가능
├── 둥글고 말랑한 실루엣 (원형 기반)
├── 큰 눈, 작은 입, 볼터치 (카와이 공식)
├── 핑크색 기본 + 컬러 배리에이션
└── 단순하지만 캐릭터성 있는 표정

     ╭─────╮
    │ ◕ ◕ │    ← 크고 반짝이는 눈
    │  ω   │    ← 작은 고양이입 (ω)
     ╰─────╯
      말랑~      ← 항상 살짝 흔들리는 idle 애니메이션
```

### 컬러 팔레트

## 코어 게임 메카닉

### 기본 규칙

```
┌─────────────────────────────────────────────────┐
│                    게임 화면                      │
│                                                 │
│     [접시1]   [접시2]   [접시3]   [접시4]         │
│     🩷🩷💛   💚💜🩷   💛💛💚   💜💜🩷          │
│                                                 │
│     → 같은 색 모찌를 같은 접시에 모으면 클리어!    │
│                                                 │
│     조작: 접시 탭 → 맨 위 모찌 선택              │
│           다른 접시 탭 → 그 위에 올려놓기         │
│           같은 색만 올릴 수 있음                  │
│                                                 │
│     [빈접시]  ← 빈 접시 1개는 임시 보관용         │
│                                                 │
│     접시가 한 색으로 가득 차면:                    │
│     → 모찌들이 "뿅!" 하고 합쳐짐                 │
│     → 큰 모찌 1개로 변신 (수집품)                 │
│     → 만족스러운 이펙트 + 사운드                  │
│                                                 │
│     ┌────────────────────────────┐              │
│     │  ⭐ +150    🪙 +30        │              │
│     │  "딸기 모찌 완성!"          │              │
│     └────────────────────────────┘              │
└─────────────────────────────────────────────────┘
```

### 난이도 진행

| 레벨 구간 | 접시 수 | 색상 수 | 빈 접시 | 특수 요소 |
|---|---|---|---|---|
| 1~20 | 3~4 | 2~3 | 2 | 없음 (튜토리얼) |
| 21~50 | 4~5 | 3~4 | 1 | 잠긴 모찌 (2번 탭해야 해제) |
| 51~100 | 5~6 | 4~5 | 1 | 얼음 모찌 (녹여야 이동 가능) |
| 101~200 | 6~7 | 5~6 | 1 | 타이머 모찌 (N턴 안에 정리) |
| 201+ | 7~8 | 6~7 | 0~1 | 복합 장애물 + 보스 주문 |

### "말랑 물리" - 핵심 차별화 요소

```
모찌의 행동:
├── 이동할 때: 늘어났다가 → 통통 튀며 착지
├── 접시에 쌓일 때: 위에서 눌리면 아래가 살짝 찌그러짐
├── 탭할 때: "몽글" 하고 눌렸다 복원
├── 클리어 시: 모찌들이 하나로 합쳐지며 "떡치기" 이펙트
├── 대기 중: 살짝살짝 흔들리는 idle 모션 (호흡하듯)
└── 실패 시: 슬픈 표정으로 바뀜 (울지 않음, 스트레스 X)

기술 구현:
├── Tween 애니메이션: scale, position, rotation 조합
├── 간단한 Shader: 찌그러짐(squash), 늘어남(stretch) 효과
├── 파티클: 합칠 때 반짝이 + 하트
└── 복잡한 물리엔진 불필요 → 전부 사전 정의 애니메이션
```

### 사운드 디자인 (ASMR 컨셉)

```
인터랙션별 사운드:
├── 모찌 탭: "뽀잉" (부드러운 탄성 사운드)
├── 모찌 이동: "슈웅~" (짧고 부드러운 스와이프)
├── 모찌 착지: "퐁!" (쿠션에 떨어지는 느낌)
├── 모찌 합체: "뿅~✨" (만족스러운 완성음)
├── 레벨 클리어: 짧은 멜로디 + 박수 (과하지 않게)
├── 코인 획득: "띠링~" (맑은 종소리)
└── 실패: 부드러운 "삐용" (불쾌하지 않은 실패음)

배경음악:
├── 메인: 로파이 피아노 + 우쿨렐레 (코지 카페 느낌)
├── 특별 스테이지: 계절별 변화 (벚꽃/여름/단풍/겨울)
└── 템포: BPM 70~90 (심장박동보다 느리게 → 이완 유도)
```

## 메타 시스템 (리텐션 장치)

### 1. 모찌 도감 (수집)

```
┌──────────────────────────────────────┐
│           🍡 모찌 도감               │
│                                      │
│  기본 모찌 (7종)                      │
│  [🩷✓] [🧡✓] [💛✓] [💚 ] [💜 ]     │
│  [🩵 ] [🤍 ]                        │
│                                      │
│  특별 모찌 (시즌 한정)                 │
│  [🎄?] [🌸?] [🎃?] [☀️?]           │
│                                      │
│  전설 모찌 (1% 확률)                  │
│  [🌈?] [⭐?] [👑?]                  │
│                                      │
│  수집률: 23/50 (46%)                 │
└──────────────────────────────────────┘
```

- 레벨 클리어 시 랜덤으로 모찌 도감에 추가
- 희귀도 시스템: 일반(70%) → 레어(20%) → 에픽(8%) → 전설(2%)
- 도감 완성 보상: 특별 테마, 접시 스킨, 배경

### 2. 떡집 꾸미기 (커스터마이징)

```
┌──────────────────────────────────────┐
│          🏪 나의 떡집                 │
│                                      │
│   ┌────────────────────────┐         │
│   │  [간판]  나의 떡집      │         │
│   │  [테이블] [의자] [화분]  │         │
│   │  [진열대] 🍡🍡🍡       │         │
│   │  [바닥]  [벽지]  [조명] │         │
│   └────────────────────────┘         │
│                                      │
│   코인으로 가구/소품 구매              │
│   배치는 자유 (드래그 앤 드롭)         │
│   시즌 한정 아이템 (벚꽃 테마 등)      │
│   친구 떡집 방문 (소셜)               │
└──────────────────────────────────────┘
```

- 퍼즐 = 코인 획득 수단, 떡집 꾸미기 = 코인 소비 목적
- 이 루프가 장기 리텐션의 핵심

### 3. 데일리 시스템

| 시스템 | 설명 | 목적 |
|---|---|---|
| 일일 주문 | 매일 3개 특별 주문 (보너스 코인) | 매일 접속 동기 |
| 연속 출석 | 7일 연속 → 특별 모찌 | 스트릭 심리 |
| 주간 도전 | "이번 주: 녹차 모찌 50개 정리" | 중기 목표 |
| 시즌 패스 | 30일 미션 → 단계별 보상 | 장기 리텐션 |

### 4. 소셜 요소 (가볍게)

- 주간 랭킹 (스코어 기반, 강제 아님)
- 친구 떡집 방문하기
- 모찌 선물하기 (하루 1회)

## 경쟁 대비 포지셔닝

```
기존 소트 퍼즐        vs        말랑소트
─────────────────────────────────────────
공/물/나사 (차갑고 무미)  →  모찌 (따뜻하고 귀여움)
무음 or 일반 효과음      →  ASMR 사운드 (만족감)
정적인 오브젝트          →  말랑말랑 물리 (생동감)
클리어 → 다음레벨 → 반복  →  클리어 → 떡집 꾸미기 → 수집
남녀 무관 디자인          →  카와이 파스텔 (타겟 명확)
레벨만 있음              →  도감 + 떡집 + 시즌 (깊이)
```
